#Introduction
The project is a static webpage clone, designed using HTML, CSS, and JavaScript. It aims to demonstrate web development skills by replicating the Netflix Payment Landing Page's visual design and interactivity.

#Technologies Used
HTML
CSS
JavaScript
#Features
#Responsive Design: The page layout adjusts seamlessly to various screen sizes and devices.
#Interactive Elements: Users can explore and select different subscription plans, with plan details displayed dynamically.
#Visually Appealing: The clone captures the visual aesthetics of the original Netflix payment page.
#Installation
Clone the repository: git clone https://github.com/your-username/Clone-Of-Netflix-Payment-Landing-Page.git
Open the index.html file in your web browser.
Usage
Explore and compare different subscription plans.
Click on a plan to view plan details and select your preferred option.
